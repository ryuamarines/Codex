# RoomPlaner

間取り図画像を下敷きにして、自由形状の部屋、窓、扉、窓前制約ゾーン、家具を 2D 上で編集し、干渉を確認するためのローカル向け Web アプリです。

## 技術スタック

- Next.js
- TypeScript
- Tailwind CSS
- Konva / React Konva
- localStorage

## できること

- png / jpg の間取り図画像を背景として読み込み
- 背景画像の表示切替、透明度調整、ロック
- 背景上の 2 点を使った縮尺設定
- 点打ちによる自由形状の部屋トレース
- 壁に沿った窓・扉の配置
- 窓前制約ゾーンの矩形配置
- 家具の追加、ドラッグ移動、サイズ変更、回転
- 家具同士 / 壁はみ出し / 扉可動範囲 / 窓前制約ゾーンの干渉警告
- マウスホイールでのズーム、ドラッグパン、グリッド表示
- グリッドスナップのオンオフと刻み変更
- CSV インポート / エクスポート
- Undo / Redo
- 家具複製
- キーボードショートカット
- 矢印キーでの微移動
- Shift+クリックによる複数選択
- 窓 / 扉 / 家具 / 制約ゾーンの寸法オーバーレイ
- ズームに応じたグリッド密度調整
- プロジェクト名の編集
- 背景画像の削除
- 部屋 / 背景へのフィット表示
- 警告内訳の表示
- `Cmd/Ctrl + 0` で表示リセット
- Planner5D の半手動移行アシスタント
- 部屋輪郭の頂点ドラッグによる後編集
- 部屋輪郭全体のドラッグ移動
- 辺の中央ハンドルからの頂点追加
- 家具 / 制約ゾーンの背景合わせ矩形生成
- 初回確認用のサンプルルーム

## 起動方法

```bash
cd RoomPlaner
npm install
npm run dev
```

ブラウザで `http://localhost:3000` を開いてください。

開発中に `Cannot find module './778.js'` や `__webpack_modules__[moduleId] is not a function` のようなエラーが出たら、`.next` キャッシュが壊れていることが多いです。次で復旧できます。

```bash
npm run dev:reset
```

## 使い方の流れ

1. 左パネルから背景画像を読み込みます。
2. `スケール設定` を選び、背景上の 2 点をクリックして実寸 mm を入力します。
3. `壁をトレース` で部屋輪郭を作ります。
4. `家具を追加` と `窓前制約ゾーン` は、始点と終点を背景に合わせてクリックすると、その矩形サイズで生成できます。
5. `窓を追加` `扉を追加` で必要な要素を置きます。
6. 家具はキャンバス上でドラッグし、右パネルでサイズや回転を調整します。
7. 選択モードではマウスホイールでズーム、キャンバスドラッグでパンできます。
8. 部屋輪郭を選択すると、輪郭全体をドラッグして背景に合わせたり、頂点をドラッグして形状を微調整できます。
9. 辺の中央に出るハンドルを押すと、その辺の途中に頂点を追加できます。
10. 必要に応じてグリッドスナップを有効にすると、配置や編集が揃えやすくなります。
11. CSV で保存して、別ブラウザや別案の比較用に読み戻せます。
12. Undo / Redo で試行錯誤を戻したりやり直したりできます。
13. 家具は複製して、近い案を並べて比較できます。
14. `Cmd/Ctrl + Z` などのショートカットで操作を素早く回せます。
15. 矢印キーで家具や制約ゾーンを微調整できます。`Shift` 付きで大きく動かせます。
16. `Shift + クリック` で複数選択し、一括で移動や削除ができます。
17. 選択中オブジェクトにはキャンバス上に寸法が出るので、目視確認しやすくなっています。
18. グリッドはズーム倍率に応じて見やすい密度に切り替わります。
19. 干渉があると家具が赤く警告表示され、右パネルにも理由が出ます。

## Planner5D からの移行

DXF などの有料 export が使えない前提で、Planner5D の画面上で確認できる情報をこのアプリへ半手動で移すための導線を入れています。

1. Planner5D の平面図スクリーンショットを保存します。
2. このアプリで背景画像として読み込みます。
3. 左パネルの `Planner5D 移行アシスタント` を開きます。
4. 壁長が読めるなら、`right 4200` のような形式で壁パスを貼り付けて部屋を生成します。
5. 家具寸法が分かるなら、`name,width,depth,x,y,rotation` の CSV を貼り付けます。
6. 窓や扉も `type,wallIndex,offsetMm,widthMm,swing,openDirection,note` 形式で追加できます。古い 6 列形式も読めます。

補足:
- 壁パスは直交部屋向けです。
- 家具の `x,y` は origin からの中心座標 mm です。
- スクリーンショット下敷きと組み合わせると、数値が足りない部分も手で合わせやすくなります。

## メモ

- 保存先はブラウザの `localStorage` です。
- `サンプルに戻す` を押すと初期データへ戻ります。
- 今回は MVP のため、1 画面編集に集中しています。プロジェクト一覧や認証、3D 機能は含んでいません。
