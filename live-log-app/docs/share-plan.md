# 共有方針

LiveLogの共有対象は、集計結果から生成したPNG画像だけです。

## 目的

- 個人用アーカイブの非公開性を崩さない
- iOS / Androidの標準共有シートからSNSやメッセージアプリへ画像を渡す
- Web Share APIに対応しない環境でもPNGを保存できるようにする

## 共有対象

- 集計タイルのPNG
- 年別まとめのPNG

共有画像には個別公演のメモやDrive画像原本を含めません。

## 動作

- `navigator.canShare({ files })`に対応するスマホでは、PNGファイルを標準共有シートへ渡す
- 非対応ブラウザやPCではPNG保存を利用する
- Xなど共有先がファイル受領に対応しない場合も、保存したPNGを投稿画面から選択できる

## 作らないもの

- 認証なしで開ける共有URL
- URLパラメータに集計データを埋め込む共有スナップショット
- 個別ライブ記録の公開ページ
- Google Drive画像原本の公開リンク
- 写真付きの公開タイムラインや公開プロフィール

共有URL関連の旧実装は削除済みです。今後も要件が明示的に変わらない限り、画像共有だけを維持します。
